5. Получение информации для команд ls и cd с помощью утилиты man:
man ls cd
man-документ состоит из следующих пунктов: имя, обзор, описание, опции POSIX, стандартные опции GNU, окружение, ошибки реализации, совместимость, смотри также, замечания.
6. Получение краткой информации по командам ls и cd с помощью команд whatis и apropos. Командные строки whatis ls и apropos cd. Различие команд whatis и apropos в том, что команда whatis позволяет искать заголовки, по целым словам, а команда apropos выполняет поиск по образцу, а не по целым словам.
7. Получение информации для команд ls и cd с помощью утилиты info:
info ls (Содержит описание утилиты.)
info cd
